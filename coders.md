## Introduction

For coders, **SOZ-LIVE** is a platform for developing additional functionality for CAD, focussing on:

- Structure
- Consistency
- Simplicity
- Modularity
- Extensibility
- Readability
- Documentation

It enables coders to use OOP techniques from within a functional / scripting language.
