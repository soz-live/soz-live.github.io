  - title: "What is an Object?"
    content: "An Object is a self contained thing.<br>
      It is an instance of a Class created at run-time and is passed and referenced as an ID.<br>
      Everything created in **SOZ** is an 'Object' of a particular Class.<br>
      This ranges from Values through to Entities such as Lines and Circles, to Parts and Assemblies.<br>
      Refer to the [Glossary](/docs/glossary.html) for additional definitions used in the project."
