## Introduction

This page contains information on the different releases of **SOZ-LIVE**.


## Details

Each of the Releases are listed below...


### CORE

This release combines the core functionality of the **soz-live** repo with the **soz-files** repo to enable data files to be imported and exported from a drawing and objects and entities to be re-used in other drawings.  


### BASE

This release builds on the **CORE** release and adds in additional classes and functionality.  
Refer to the Table below and the links to the included repos for further details.


### AEC

This release builds on the **BASE** release and adds in additional classes and functionality for AEC.
Refer to the Table below and the links to the included repos for further details.


### RIG

This release builds on the **AEC** release and adds in additional classes and functionality for Rigging Iteams and Assemblies associated with Crane rigging arrangements.  
Refer to the Table below and the links to the included repos for further details.


### ALL

This cumulative release includes all released classes and functionality.  
Curently this release has the same functionality as the **RIG** release.

### DEV

This release builds on the **ALL** release and adds in additional classes and functionality under development and for Developers. 
Refer to the Table below and the links to the included repos for further details.
